# Evidence classes

- **Exact arithmetic:** rational or integer computations with no tolerance.
- **Validated interval computation:** outward-rounded enclosures and exhaustive subdivision used as proof-bearing evidence.
- **Independent reconstruction:** a separately written implementation recomputing selected quantities from canonical inputs.
- **Floating-point corroboration:** useful for reconstruction and diagnostics but not the primary source of a certified claim.
- **Recorded search output:** data from the documented study environment; not a platform-independent enumerative theorem.
- **Open:** a stated question for which no proof is asserted.

The checker fails closed on malformed inputs and corrupted obligations in the default CI smoke suite. The complete adversarial release suite additionally probes ambiguous intervals, incomplete partitions, and rounding knife edges.
